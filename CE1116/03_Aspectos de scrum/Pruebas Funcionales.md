## 📚 Idea/Concepto 

Es una prueba de software que verifica si el producto realizado cumple con las funcionalidades pedidas por el cliente. Estas son esenciales para el control de calidad del producto. Son basados en la experiencia de usuario según los requisitos especificados. 
## 📌 Puntos Claves (Opcional)
- Pruebas ad hoc: intuición de testers
- Pruebas de API
- Pruebas de integración
- Pruebas de regresión
- Pruebas del sistema
- Pruebas de aceptación del usuario
- Etc
