## 📚 Idea/Concepto 

Las redes neuronales son funciones matemáticas de las IAs para procesar datos, de manera similar a las capas neuronales. Se tienen múltiples capas ocultas para modelar relaciones no lineales complejas. Se detectan los patrones de los datos usando pesos y sesgos. El procesamiento de datos ocurre transformando las entradas en salidas mediante un proceso de entrenamiento.
## 📌 Puntos Claves (Opcional)
- La forma en que se conectan las neuronas y organizan capas define el tipo de red
- Aprenden reglas y patrones
- Con el entrenamiento minimiza la función de perdida
